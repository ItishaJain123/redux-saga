Installation & Setup

Redux Saga is added to a Redux application as a middleware.

The setup process is straightforward and happens in three steps.

Step 1: Install the package

  


npm install redux-saga

This installs the core Redux Saga library.

Step 2: Create a saga middleware

Redux Saga works by creating a middleware instance and attaching it to the Redux store. 

  


import createSagaMiddleware from "redux-saga";

const sagaMiddleware = createSagaMiddleware();

Step 3: Add middleware to the store

The saga middleware is added when creating the Redux store.

  


import { configureStore } from "@reduxjs/toolkit";

const store = configureStore({
  reducer: rootReducer,
  middleware: (getDefaultMiddleware) =>
    getDefaultMiddleware().concat(sagaMiddleware),
});

Step 4: Run the root saga

After creating the store, start the saga by running the root saga.

  


import rootSaga from "./sagas";

sagaMiddleware.run(rootSaga);

Key takeaway

Redux Saga setup happens once. After that, all async logic lives inside sagas, not inside components or reducers.
